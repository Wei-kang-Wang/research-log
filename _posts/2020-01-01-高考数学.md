### 1. **\[2026 湖南九校联盟第二次联考 19题\]**

设函数$$f(x) = x^3 -ax -b, x \in \mathbb{R}, a,b \in \mathbb{R}$$
* (1) 讨论函数$$f(x)$$的单调区间
* (2) 当$$a = 2, b = 0$$时，函数$$f(x)$$的图像上有且仅有两个点到原点的距离为$$d$$，求$$d$$的取值范围
* (3) 函数$$f(x)$$的图象上是否存在唯一的一组点$$A_1, A_2, \cdots, A_n$$，构成正$$n$$边形且$$n \geq 3$$？若存在，请求出所有满足条件的$$n$$以及对应的$$a$$的值，若不存在，请说明理由。

**解答**
* (1) $$f^{'}(x) = 3x^2 - a$$，当$$a < 0$$时，$$f^{'}(x)$$在$$\mathcal{R}$$上均为正，从而$$f(x)$$在$$\mathcal{R}$$上单调递增。当$$a geq 0$$时，$$f(x)$$在$$(- \infty, -\sqrt{a/3}) \bigcup (\sqrt{a/3}, \infty)$$上单调递增，在$$(-\sqrt{a/3}, \sqrt{a/3})$$上单调递减。
* (2) 当$$a=2, b=0$$时，$$f(x) = x^3 - 2x$$，从而$$f(x)$$上任意一点$$(x, f(x))$$到原点的距离$$d = \sqrt{x^2 + (x^3 - 2x)^2} = \sqrt{x^6 - 4x^4 + 5x^2}$$。设$$t = x^2$$，如果$$x=0$$，则$$d=0$$，而$$\sqrt{x^6 - 4x^4 + 5x^2} = 0 \Leftrightarrow x^6 - 4x^4 + 5x^2 = 0  \Leftrightarrow x^2(x^4 - 4x^2 + 5) =0$$，只有$$x=0$$一个解，不符合要求，从而$$t>0$$，从而$$d = \sqrt{x^6 - 4x^4 + 5x^2}$$对于$$x$$有两个解，等价于$$d = \sqrt{t^3 - 4t^2 + 5t}$$对于$$t$$仅有一个解，即$$t^3 - 4t^2 + 5t - d^2=0$$仅有一个解，设$$g(t) = t^3 - 4t^2 + 5t - d^2$$，那么$$g^{'}(t) = 3t^2 - 8t + 5 = (3t-5)(t-1)$$，其在$$(- \infty,1 ) \bigcup (5/3, \infty)$$上单调递增，在$$(1, 5/3)$$上单调递减，而$$g(t) \rightarrow \infty, x \rightarrow \infty$$以及$$g(t) \rightarrow -\infty, x \rightarrow -\infty$$，从而$$g(1)<0$$或者$$g(5/3)>0$$满足$$g(t)=0$$只有一个解，即$$d^2>2$$或者$$d^2 < 50/27$$，又因为$$d > 0$$，从而$$d \in (0, 5\sqrt{6}/9) \bigcup (\sqrt{2}, \infty)$$。
* (3) 首先，$$b$$只会让$$f(x)$$的图像沿着$$y$$轴上下平移，不改变$$f(x)$$在各点的曲率，从而不妨假设$$b=0$$，即考虑$$f(x) = x^3 - ax$$，此时$$f(x)$$是一个奇函数，那么如果存在一个不按照原点中心对称的正多边形，我们其中心对称图形也一定位于$$f(x)$$上，且和原正多边形不重合，这和唯一性矛盾，从而如果存在该正多边形，我们只需要考虑按照原点中心对称的正多边形。而且如果$$n$$是奇数，那么正$$n$$边形是无法按照原点中心对称的，从而$$n$$必然是偶数。从而我们可以知道，$$A_1, A_2, \cdots, A_n$$必然是某个以原点为中心的圆和$$f(x)$$的交点，假设该圆的方程为$$x^2 + y^2 = R^2$$，那么$$A_1, A_2, \cdots, A_n$$的坐标，都是方程$$x^2 + (x^3 - ax)^2 = R^2$$的解，这是一个关于$$x$$的六次方程，最多有六个解，从而$$n \leq 6$$，结合$$n$$是偶数，从而$$n=4,6$$。
* * 当$$n=4$$时，假设其对角两点构成的直线方程分别为$$y=kx, y=-1/k x$$，即$$A_1A_3$$和$$A_2A_4$$，联立$$f(x) = x^3 - ax$$，可得：$$x^3 - ax = kx$$，即$$x^2 = a+k$$是点$$A_1, A_3$$横坐标的解，同理$$x^2 = a-1/k$$是点$$A_2, A_4$$的解。题目要求这样的正多边形是唯一的，从而因为$$A_1O = A_2O=A_3O=A_4O$$，可得$$\sqrt{1 + k^2}\sqrt{a+k} = \sqrt{1+1/k^2}\sqrt{a-1/k}$$对于$$k$$有唯一的解，即$$(k^2-1)a + k^3 + 1/k=0$$对于$$k$$有唯一的解。$$k=\pm 1$$显然不满足要求，从而$$a = -\frac{k^3 + 1/k}{k^2 - 1} = -\frac{k^2 + 1/k^2}{k - 1/k} = - \frac{(k-1/k)^2 + 2}{k-1/k} = -[(k-\frac{1}{k}) + \frac{2}{k - \frac{1}{k}}]$$，如果该关于$$k$$的方程只有一个解，根据均值不等式，只有$$a = -2 \sqrt{2}$$，此时$$k - 1/k = \sqrt{2}$$，即$$k, -1/k = \frac{\sqrt{2} \pm \sqrt{6}}{2}$$满足条件。
  * 当$$n=6$$时，假设$$A_1$$的坐标是$$(x,y)$$，那么按照逆时针$$A_2, A_3, A_4, A_5, A_6$$的坐标分别为：$$(\frac{x+\sqrt{3}y}{2}, \frac{-\sqrt{3}x+y}{2}), (\frac{-x+\sqrt{3}y}{2}, \frac{-\sqrt{3}x-y}{2}), (y, -x), (\frac{-x-\sqrt{3}y}{2}, \frac{\sqrt{3}x-y}{2}), (\frac{x-\sqrt{3}y}{2}, \frac{\sqrt{3}x+y}{2})$$，将$$A_1, A_2, A_3$$的坐标和$$f(x) = x^3 - ax$$联立，可得：$$y = x^3 - ax, \frac{-\sqrt{3}x + y}{2} = (\frac{x + \sqrt{3}y}{2})^3 - a \frac{x + \sqrt{3}y}{2},  \frac{-\sqrt{3}x - y}{2} = (\frac{-x + \sqrt{3}y}{2})^3 - a \frac{-x + \sqrt{3}y}{2}$$，第二个式子和第三个式子相减，得到$$y = \frac{x^3 + 3y^2 x}{4} - ax$$，再和第一个式子联立，得到$$y^2x = x^3$$，显然$$x \neq 0$$，且该图形为中心对称图形，从而不妨假设$$A_1$$的坐标为$$(x,x)$$。将表示$$A_2$$和$$A_3$$的式子相加，得到$$x^2 = \frac{2a-2}{3}$$，而从$$A_4$$坐标可得：$$x^2 = a-1$$，从而$$a -1 = \frac{2a-2}{3}$$，只有$$a=1$$这一个解，但是此时$$x=0$$不满足要求。从而正六边形不存在。
* 综上所述，$$n=4, a=2\sqrt{2}$$。
