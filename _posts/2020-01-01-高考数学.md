### 1. **\[2026 湖南九校联盟第二次联考 19题\]**

设函数$$f(x) = x^3 -ax -b, x \in \mathbb{R}, a,b \in \mathbb{R}$$
* (1) 讨论函数$$f(x)$$的单调区间
* (2) 当$$a = 2, b = 0$$时，函数$$f(x)$$的图像上有且仅有两个点到原点的距离为$$d$$，求$$d$$的取值范围
* (3) 函数$$f(x)$$的图象上是否存在唯一的一组点$$A_1, A_2, \cdots, A_n$$，构成正$$n$$边形且$$n \geq 3$$？若存在，请求出所有满足条件的$$n$$以及对应的$$a$$的值，若不存在，请说明理由。

**解答**
* (1) $$f^{'}(x) = 3x^2 - a$$，当$$a < 0$$时，$$f^{'}(x)$$在$$\mathcal{R}$$上均为正，从而$$f(x)$$在$$\mathcal{R}$$上单调递增。当$$a \geq 0$$时，$$f(x)$$在$$(- \infty, -\sqrt{a/3}) \bigcup (\sqrt{a/3}, \infty)$$上单调递增，在$$(-\sqrt{a/3}, \sqrt{a/3})$$上单调递减。
* (2) 当$$a=2, b=0$$时，$$f(x) = x^3 - 2x$$，从而$$f(x)$$上任意一点$$(x, f(x))$$到原点的距离$$d = \sqrt{x^2 + (x^3 - 2x)^2} = \sqrt{x^6 - 4x^4 + 5x^2}$$。设$$t = x^2$$，如果$$x=0$$，则$$d=0$$，而$$\sqrt{x^6 - 4x^4 + 5x^2} = 0 \Leftrightarrow x^6 - 4x^4 + 5x^2 = 0  \Leftrightarrow x^2(x^4 - 4x^2 + 5) =0$$，只有$$x=0$$一个解，不符合要求，从而$$t>0$$，从而$$d = \sqrt{x^6 - 4x^4 + 5x^2}$$对于$$x$$有两个解，等价于$$d = \sqrt{t^3 - 4t^2 + 5t}$$对于$$t$$仅有一个解，即$$t^3 - 4t^2 + 5t - d^2=0$$仅有一个解，设$$g(t) = t^3 - 4t^2 + 5t - d^2$$，那么$$g^{'}(t) = 3t^2 - 8t + 5 = (3t-5)(t-1)$$，其在$$(- \infty,1 ) \bigcup (5/3, \infty)$$上单调递增，在$$(1, 5/3)$$上单调递减，而$$g(t) \rightarrow \infty, x \rightarrow \infty$$以及$$g(t) \rightarrow -\infty, x \rightarrow -\infty$$，从而$$g(1)<0$$或者$$g(5/3)>0$$满足$$g(t)=0$$只有一个解，即$$d^2>2$$或者$$d^2 < 50/27$$，又因为$$d > 0$$，从而$$d \in (0, 5\sqrt{6}/9) \bigcup (\sqrt{2}, \infty)$$。
* (3) 首先，$$b$$只会让$$f(x)$$的图像沿着$$y$$轴上下平移，不改变$$f(x)$$在各点的曲率，从而不妨假设$$b=0$$，即考虑$$f(x) = x^3 - ax$$，此时$$f(x)$$是一个奇函数，那么如果存在一个不按照原点中心对称的正多边形，我们其中心对称图形也一定位于$$f(x)$$上，且和原正多边形不重合，这和唯一性矛盾，从而如果存在该正多边形，我们只需要考虑按照原点中心对称的正多边形。而且如果$$n$$是奇数，那么正$$n$$边形是无法按照原点中心对称的，从而$$n$$必然是偶数。从而我们可以知道，$$A_1, A_2, \cdots, A_n$$必然是某个以原点为中心的圆和$$f(x)$$的交点，假设该圆的方程为$$x^2 + y^2 = R^2$$，那么$$A_1, A_2, \cdots, A_n$$的坐标，都是方程$$x^2 + (x^3 - ax)^2 = R^2$$的解，这是一个关于$$x$$的六次方程，最多有六个解，从而$$n \leq 6$$，结合$$n$$是偶数，从而$$n=4,6$$。
  * 当$$n=4$$时，假设其对角两点构成的直线方程分别为$$y=kx, y=-1/k x$$，即$$A_1A_3$$和$$A_2A_4$$，联立$$f(x) = x^3 - ax$$，可得：$$x^3 - ax = kx$$，即$$x^2 = a+k$$是点$$A_1, A_3$$横坐标的解，同理$$x^2 = a-1/k$$是点$$A_2, A_4$$的解。题目要求这样的正多边形是唯一的，从而因为$$A_1O = A_2O=A_3O=A_4O$$，可得$$\sqrt{1 + k^2}\sqrt{a+k} = \sqrt{1+1/k^2}\sqrt{a-1/k}$$对于$$k$$有唯一的解，即$$(k^2-1)a + k^3 + 1/k=0$$对于$$k$$有唯一的解。$$k=\pm 1$$显然不满足要求，从而$$a = -\frac{k^3 + 1/k}{k^2 - 1} = -\frac{k^2 + 1/k^2}{k - 1/k} = - \frac{(k-1/k)^2 + 2}{k-1/k} = -[(k-\frac{1}{k}) + \frac{2}{k - \frac{1}{k}}]$$，如果该关于$$k$$的方程只有一个解，根据均值不等式，只有$$a = -2 \sqrt{2}$$，此时$$k - 1/k = \sqrt{2}$$，即$$k, -1/k = \frac{\sqrt{2} \pm \sqrt{6}}{2}$$满足条件。
  * 当$$n=6$$时，假设$$A_1$$的坐标是$$(x,y)$$，那么按照逆时针$$A_2, A_3, A_4, A_5, A_6$$的坐标分别为：$$(\frac{x+\sqrt{3}y}{2}, \frac{-\sqrt{3}x+y}{2}), (\frac{-x+\sqrt{3}y}{2}, \frac{-\sqrt{3}x-y}{2}), (y, -x), (\frac{-x-\sqrt{3}y}{2}, \frac{\sqrt{3}x-y}{2}), (\frac{x-\sqrt{3}y}{2}, \frac{\sqrt{3}x+y}{2})$$，将$$A_1, A_2, A_3$$的坐标和$$f(x) = x^3 - ax$$联立，可得：$$y = x^3 - ax, \frac{-\sqrt{3}x + y}{2} = (\frac{x + \sqrt{3}y}{2})^3 - a \frac{x + \sqrt{3}y}{2},  \frac{-\sqrt{3}x - y}{2} = (\frac{-x + \sqrt{3}y}{2})^3 - a \frac{-x + \sqrt{3}y}{2}$$，第二个式子和第三个式子相减，得到$$y = \frac{x^3 + 3y^2 x}{4} - ax$$，再和第一个式子联立，得到$$y^2x = x^3$$，显然$$x \neq 0$$，且该图形为中心对称图形，从而不妨假设$$A_1$$的坐标为$$(x,x)$$。将表示$$A_2$$和$$A_3$$的式子相加，得到$$x^2 = \frac{2a-2}{3}$$，而从$$A_4$$坐标可得：$$x^2 = a-1$$，从而$$a -1 = \frac{2a-2}{3}$$，只有$$a=1$$这一个解，但是此时$$x=0$$不满足要求。从而正六边形不存在。
综上所述，$$n=4, a=2\sqrt{2}$$。


### 2. **\[2025 新高考一卷 19题 \]**

设函数$$f(x) = 5cosx - cos5x$$
* (1) 求$$f(x)$$在$$(0, \frac{\pi}{4})$$上的最大值
* (2) 给定$$\theta \in (0, \pi)$$，设$$a$$为实数，证明：存在$$y \in [a - \theta, a + \theta]$$，使得$$cos y \leq cos \theta$$
* (3) 若存在$$t$$使得对任意$$x$$，都有$$5cos x - cos(5x+t) \leq b$$，求$$b$$的最小值

**解答**
* (1) $$f^{'}(x) = -5sinx + 5sin5x = 5(sin5x - sinx) = 10cos3x sin2x$$，其在$$(0, \frac{\pi}{6})$$上单调递增，在$$(\frac{\pi}{6}, \frac{\pi}{4})$$上单调递减，从而最大值为$$f(\frac{\pi}{6}) = 3 \sqrt{3}$$
* (2) 考虑单位圆，那么$$cos x$$就是以$$x$$为逆时针角度的单位圆上的点在$$x$$轴上的投影。区间$$[a - \theta, a + \theta]$$一定与单位圆上除了$$[- \theta, \theta]$$的区间有交集，而再该区间上，$$cos x$$的值均小于$$[- \theta, \theta]$$区间上的值，即证。
* (3) 要求$$b$$的最小值，即要求$$\min_{t} \max_{x} 5cos x - cos(5x+t)$$，而函数$$g(x) = 5cosx - cos(5x+t)$$，其导数为$$g^{'}(x) = -5sinx + 5sin(5x+t)$$，零点为$$5x+t = x + 2k\pi$$或者$$5x+t = 2k\pi + \pi - x$$，分别求得$$x_1 = -\frac{t}{4} + \frac{k_1\pi}{2}, x_2 = -\frac{t}{6} + \frac{k_2 \pi}{3}$$，其中$$k_1, k_2$$为任意整数。对于$$x_1$$可以看到，在单位圆内将圆划分为$$\left[-\frac{\pi}{4}, \frac{\pi}{4}\right], \left[\frac{\pi}{4}, \frac{3\pi}{4}\right], \left[\frac{3\pi}{4}, \frac{-3\pi}{4}\right], \left[\frac{-3\pi}{4}, \frac{-\pi}{4}\right]$$四个区域，因为$$x_1$$的取值之间至少间隔一个$$\frac{\pi}{2}$$，所以至少有一个$$x_1$$的值落在$$\left[\frac{-\pi}{4}, \frac{\pi}{4}\right]$$内，而在该区间内，$$g(x_1) = 5cos x_1  - cos(5x_1 + t) = 5cos x_1 - cos x_1 = 4 cos x_1 \geq 2 \sqrt{2}$$，从而$$g(x)$$的最大值不小于$$2 \sqrt{2}$$。类似的，将单位圆划分为$$\left[-\frac{\pi}{6}, \frac{\pi}{6}\right], \left[\frac{\pi}{6}, \frac{\pi}{2}\right], \left[\frac{\pi}{2}, \frac{5\pi}{6}\right], \left[\frac{5\pi}{6}, \frac{-5\pi}{6}\right], \left[\frac{-5\pi}{6}, \frac{-\pi}{2}\right], \left[\frac{-\pi}{2}, \frac{-\pi}{6}\right]$$六个区域，那么因为$$x_2$$的取值之间至少间隔一个$$\frac{\pi}{3}$$，所以至少有一个$$x_2$$落在$$\left[-\frac{\pi}{6}, \frac{\pi}{6}\right]$$内，而该区间内$$g(x_2) = 5 cos x_2 - cos (5 x_2 + t) = 6cos x_2 \geq 3 \sqrt{3}$$，从而，对于任意的$$t$$，关于$$x$$的函数$$5cos x - cos(5x+t)$$最大值至少大于$$3 \sqrt{3}$$。现在证明，如果取$$t = 2k \pi$$，那么$$5 cos x - cos(5x +t)$$的最大值就是$$3 \sqrt{3}$$。在这种情况下，$$5 cos x - cos(5x +t) = 5cosx - cos5x$$，其导数的零点为$$x_1 = \frac{k_1 \pi}{2}$$或者$$x_2 = \frac{\pi}{3} + \frac{k_2 \pi}{3}$$，其中$$k_1, k_2$$是整数，验证可得，此时原函数的最大值就是$$3 \sqrt{3}$$，从而$$b$$的最小值即为$$3 \sqrt{3}$$。


### 3. **\[2013 安徽高考 21题 \]**

某高校数学系计划在周六和周日各举行一次主体不同的心理测试活动，分别由李老师和张老师负责，已知该系共有$$n$$位学生，每次活动均需该系$$k$$位学生参加（$$n$$和$$k$$都是固定的正整数）。假设李老师和张老师分别将各自活动通知的信息独立、随机的发给该系$$k$$位学生，且所发信息都能收到，记该系收到李老师或张老师所发活动通知信息的学生人数为$$X$$。

* (1) 求该系学生甲收到李老师或张老师所发活动通知信息的概率
* (2) 求使$$P(X=m)$$取得最大值的整数$$m$$

**解答**
* (1) $$p = 1 - \frac{C_{n-1}^{k}}{C_n^k} \frac{C_{n-1}^{k}}{C_n^k} = 1 - \frac{(n-k)^2}{n^2}$$
* (2) 首先，$$m$$的范围是$$k \leq m \leq \min \lbrace 2k, n \rbrace$$。当$$X=m$$时，将这$$m$$个人分为三组：只收到李老师的通知，有$$m-k$$人，只收到张老师的通知，也是$$m-k$$人，同时收到了两个老师的通知，有$$2k-m$$人，所以事件$$X=m$$的基本事件数为$$C_n^k C_{k}^{2k-m}C_{n-k}^{m-k}$$，即李老师先从$$n$$个学生里选$$k$$个，再从这$$k$$个学生里找$$2k-m$$个是张老师和李老师都通知的，最后再在剩下的$$n-k$$个里找到$$m-k$$个没有被张老师通知的。总事件数为$$(C_n^k)^2$$个。
  * 如果$$k=n$$，那么$$m$$只能取$$n$$，此时$$P(X=m)=1$$
  * 如果$$k < n$$，那么$$P(X=m) = \frac{C_n^k C_{k}^{2k-m}C_{n-k}^{m-k}}{(C_n^k)^2} = \frac{C_k^{2k-m} C_{n-k}^{m-k}}{C_n^k} = \frac{C_k^{m-k} C_{n-k}^{m-k}}{C_n^k}$$，我们要求$$P(X=m)$$的最大值，因为我们要求的是正整数的最大值，其区别于连续值的最大值，$$P(X=m)$$取最大值的一个必要条件是$$P(X=m-1) \leq P(X=m) \geq P(X=m+1)$$，而$$\frac{P(X=m+1)}{P(X=m)} = \frac{C_k^{m+1-k}C_{n-k}^{m+1-k}}{C_k^{m-k}C_{n-k}^{m-k}} = \frac{(n-m)(2k-m)}{(m+1-k)^2} \leq 1$$，等价于$$2kn - (k-1)^2 - (n+2)m \leq 0$$，解得$$m \geq 2k - \frac{(k+1)^2}{n+2}$$。需要验证$$m$$是否满足$$k \leq m \leq \min \lbrace 2k, n \rbrace$$。构建关于$$x$$的一次函数$$f(x) = -(n+2)x + 2kn - (k-1)^2$$，$$f(k) = kn - k^2 - 1 \geq k(k+1) - k^2 - 1 = k -1 \geq 0$$，$$f(2k) = -4k - (k-1)^2 < 0$$，$$f(n) = -n^2 - 2n + 2kn - (k-1)^2 = -(n-k+1)^2 \leq 0$$，即对于该一次函数$$f(x)$$，$$x$$取$$k$$和$$x$$取$$2k$$或者$$n$$的符号不同，又因为$$f(2k - \frac{(k+1)^2}{n+2})=0$$，从而$$2k - \frac{(k+1)^2}{n+2}$$一定在区间$$(k, \min \lbrace 2k, n \rbrace)$$内，验证完毕。另一个方向是$$\frac{P(X=m)}{P(X=m-1)} \geq 1$$，而其计算过程和上面完全相同，直接可以得到结果是$$m \leq 2k+1 - \frac{(k+1)^2}{n+2}$$。从而我们得到$$2k - \frac{(k+1)^2}{n+2} \leq m \leq 2k+1 - \frac{(k+1)^2}{n+2}$$，如果$$\frac{(k+1)^2}{n+2}$$是一个整数，那么$$m = 2k - \frac{(k+1)^2}{n+2}$$或者$$m = 2k - \frac{(k+1)^2}{n+2} + 1$$。如果$$\frac{(k+1)^2}{n+2}$$不是整数，那么$$m = 2k - \lfloor \frac{(k+1)^2}{n+2} \rfloor$$。


### 4. **\[2025 四省联考 9题\]**

已知双曲线$$C: \frac{x^2}{a^2} - \frac{y^2}{b^2} = 1, a>0, b>0$$的离心率为$$\frac{2\sqrt{3}}{3}$$，且经过点$$P(2,1)$$

* (1) 求$$C$$的方程
* (2) 作$$C$$在点$$P$$处的切线$$I$$，设$$I$$与$$C$$的两条渐近线分别交于$$Q,R$$，求$$\lvert QR \rvert$$
* (3) 将横纵坐标均为正整数的点称为格点，记$$C$$上所有格点为$$P_1(x_1, y_1), P_2(x_2, y_2), \cdots$$，且$$x_1 < x_2 < \cdots$$，证明$$\frac{x_{n+2} + x_n}{x_{n+1}}$$是定值

**解答**
* (1) 离心率$$\frac{2\sqrt{3}}{3} = \frac{\sqrt{a^2+b^2}}{a}$$得到，$$a^2 = 3b^2$$，由$$C$$经过点$$P$$得到，$$\frac{4}{a^2}-\frac{1}{b^2}=1$$，从而$$a^2 = 1, b^2 = 1/3$$，$$C$$的方程为$$x^2 - 3y^2 = 1$$
* (2) 对$$C$$隐函数求导，得到$$2x - 6yy^{'} = 0$$，从而$$C$$在$$P$$点的导数为$$y^{'} = 2/3$$，从而$$P$$点处的切线方程为$$y-1 = 2/3(x-2)$$。而$$C$$的渐近线方程为：$$y=\pm \frac{1}{\sqrt{3}}x$$，将其与切线方程联立，得到$$Q,R$$的横坐标为$$2\pm \sqrt{3}$$，从而$$\lvert QR \rvert = \frac{2\sqrt{39}}{3}$$
* (3) 首先，很容易看出来$$x=2, y=1$$是方程的正整数解，而且很容易验证是最小的正整数解。记$$x_1=2, y_1=1$$，那么$$(x_1 + \sqrt{3} y_1)(x_1 - \sqrt{3}y_1) = 1$$，从而对于任意的正整数$$n$$，$$(x_1 + \sqrt{3} y_1)^n (x_1 - \sqrt{3} y_1)^n = 1$$，而且可以注意到，对于正整数$$x_1, y_1$$，$$(x_1 + \sqrt{3} y_1)^n$$和$$(x_1 - \sqrt{3} y_1)^n$$的展开分别是$$x_n + \sqrt{3} y_n$$以及$$x_n - \sqrt{3} y_n$$的形式，其中$$x_n, y_n$$也是正整数，从而$$x_n^2 - 3y_n^2 = 1$$，即$$(x_n, y_n)$$也是原方程的一组正整数解。下面我们只需要说明这种构造即得到了原方程所有的整数解即可。利用反证法，假设对于某个$$n$$，存在一组正整数$$(x_0, y_0)$$，满足$$x_n < x_0 < x_{n+1}$$，且$$x_0^2 - 3y_0^2 = 1$$，那么$$(x_0 - \sqrt{3} y_0) (x_0 + \sqrt{3} y_0) = 1$$，又有$$(x_n - \sqrt{3} y_n) (x_n + \sqrt{3} y_n) = 1$$，两个式子相乘，得到：$$((x_0x_n - 3y_0y_n) + \sqrt{3}(x_ny_0 - x_0y_n))((x_0x_n - 3y_0y_n) - \sqrt{3}(x_ny_0 - x_0y_n)) = 1$$，而因为$$x_0, x_n$$是原方程的解，从而$$x_0x_n - 3y_0y_n>0$$，因为$$x_0 > x_n$$，从而$$x_ny_0 - x_0y_n >0$$。又因为$$x_0x_n - 3y_0y_n, x_ny_0 - x_0y_n$$是整数，从而它们为正整数。从而由之间的式子，$$(x_0x_n - 3y_0y_n, x_ny_0 - x_0y_n)$$也是原方程的一个正整数解。根据我们的假设$$x_n <x_0 < x_{n+1}$$，从而$$y_n < y_0 < y_{n+1}$$，从而$$x_n + \sqrt{3} y_n < x_0 + \sqrt{3} y_n < x_{n+1} + \sqrt{3} y_{n+1}$$，而由我们的构造，$$x_n + \sqrt{3} y_n = (2 + \sqrt{3})^n$$，从而$$1 < \frac{x_0 + \sqrt{3} y_0}{x_n + \sqrt{3} y_n} < 2+\sqrt{3}$$，而$$\frac{x_0 + \sqrt{3} y_0}{x_n + \sqrt{3} y_n} = \frac{(x_0 + \sqrt{3} y_0)(x_n - \sqrt{3} y_n)}{(x_n + \sqrt{3} y_n)(x_n - \sqrt{3} y_n)} = (x_0 + \sqrt{3} y_0)(x_n - \sqrt{3} y_n) = x_0x_n - 3y_0y_n + \sqrt{3}(x_ny_0 - x_0y_n)$$，从而$$1 < x_0x_n - 3y_0y_n + \sqrt{3}(x_ny_0 - x_0y_n) < 2+ \sqrt{3}$$，由之前的推导可知，$$x_0x_n - 3y_0y_n, x_ny_0 - x_0y_n$$均是正整数，且$$x_0$$不是第一个解，即$$x_0 > x_1 = 2$$，从而$$ x_0x_n - 3y_0y_n + \sqrt{3}(x_ny_0 - x_0y_n) > 2+\sqrt{3}$$，矛盾。即原方程不存在上述构造的$$x_n, y_n$$以外的正整数解。而$$x_n = \frac{1}{2}((2+ \sqrt{3})^n + (2- \sqrt{3})^n)$$，从而因为$$((2+\sqrt{3})^{n+1} + (2-\sqrt{3})^{n+1})((2+\sqrt{3})+ (2-\sqrt{3})) = (2+\sqrt{3})^{n+2} + (2-\sqrt{3})^{n+2} + (2+\sqrt{3})^{n} + (2-\sqrt{3})^{n}$$，即$$4x_{n+1} = x_{n+2} + x_n$$，从而$$\frac{x_{n+2} + x_n}{x_{n+1}} = 4$$。


### 5. **\[2022 浙江高考\]**

已知函数$$f(x) = \frac{e}{2x} + ln x (x>0)$$，$$(x_1, f(x_1)), (x_2, f(x_2)), (x_3, f(x_3))$$为函数图像$$y=f(x)$$上的三点，且它们的切线均经过$$(a,b)$$。

* (1) 求$$f(x)$$的单调区间
* (2) 若$$a > e$$，证明：$$0 < b - f(a) < \frac{1}{2}(\frac{a}{e} - 1)$$
* (3) 若$$0 < a < e$$，且$$x_1 < x_2 < x_3$$，证明：$$\frac{2}{e} + \frac{e-a}{6e^2} < \frac{1}{x_1} + \frac{1}{x_3} < \frac{2}{a} - \frac{e-a}{6e^2}$$

**解答**
* (1) $$f^{'}(x) = \frac{2x - e}{2x^2}$$，从而$$f(x)$$在$$(0, e/2)$$上单调递减，在$$(e/2, \infty)$$上单调递增
* (2) 由题目条件可知，$$a,b$$需要满足的条件即是过点$$(a,b)$$可以作三条函数$$f(x)$$的切线，即$$f(x) - b = f^{'}(x)(x-a)$$有三个不同的解，即$$lnx + \frac{e+a}{x} - \frac{ae}{2x^2} - b - 1 = 0$$有三个不同的解。记$$g(x) = lnx + \frac{e+a}{x} - \frac{ae}{2x^2} - b - 1$$，那么$$g^{'} = \frac{(x-a)(x-e)}{x^3}$$，因为$$a>e$$，从而$$g(x)$$在$$(0,e) \bigcup (a, \infty)$$上单调递增，在$$(e,a)$$上单调递减，从而$$g(x)=0$$有三个不同的解，$$g(e)>0>g(a)$$，即$$g(a) = lna + \frac{a+e}{a} - \frac{e}{2a} - b - 1 <0$$，且$$g(e) = \frac{a+e}{e} - \frac{a}{2e} - b > 0$$，即$$lna + \frac{e}{2a} - b < 0$$，$$\frac{a}{2e} - b + 1 >0$$。从而$$b-f(a) = b - lna - \frac{e}{2a}>0$$，不等式左边得证。且$$b - f(a) = b - lna - \frac{e}{2a} < 1 + \frac{a}{2e} - lna - \frac{e}{2a}$$，如果要证明$$b - f(a) < \frac{a}{2e} - \frac{1}{2}$$，只需要证明$$1 + \frac{a}{2e} - lna - \frac{e}{2a} < \frac{a}{2e} - \frac{1}{2}$$，即$$lna + \frac{e}{2a}>\frac{2}{3}$$。由(1)可知，$$f(x)$$在$$(e/2, \infty)$$上单调递增，从而$$f(a) > f(e) = \frac{2}{3}$$，即证。
* (3) 首先进行倒数换元，令$$t = 1/x$$，从而$$g(x)$$即转换为$$h(t) = -lnt + (e+a)t - 2aet^2 - b - 1$$，由(2)中的结论，$$h(t) = 0$$的三个解满足$$t_3 < \frac{1}{e} < t_2 < \frac{1}{a} < t_1$$，我们要证明的即为$$\frac{2}{e} + \frac{e-a}{6e^2} < t_1 + t_3 < \frac{2}{a} - \frac{e-a}{6e^2}$$，等价于$$(t_1+t_3 - (\frac{2}{e} + \frac{e-a}{6e^2}))(t_1+t_3 - (\frac{2}{a} - \frac{e-a}{6e^2}))<0$$，即$$\frac{ea}{2}(t_1+t_3)^2 - (a+e)(t_1+t_3) + \frac{ea}{2}(\frac{2}{e} + \frac{e-a}{6e^2})(\frac{2}{a} -
